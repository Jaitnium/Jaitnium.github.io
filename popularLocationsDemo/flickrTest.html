About

About this project:
The ultimate goal of this project is create a webpage that will show the most popular locations to travel to around the world. You'll be able to enter in a city from any country at any time of the year and know what places you should visit and what local events are happening.

Technical Overview:
This project uses flickr to obtain hundreds of thousands of picture locations. Each picture has a latitude, longitude, and date in which it was taken. A local python script uses the flickr api to create JSON files of picture locations. The user specifies a month of a given year and then what city they want displayed. The corresponding JSON file is read and the locations are mapped using the google maps api. A heatmap layer is then applied to the map so it's easier to see where the most activity is.
